
#  Suicide Prevention Dashboard – France

##  Project Description

This project is an interactive data visualization dashboard focused on **suicide prevention in France**.

It is designed for **university students** and follows a **storytelling approach**, guiding users through key questions such as:

* Are some groups more affected than others?
* What role do age, gender, and socio-economic factors play?
* Are there regional differences?

The dashboard aims to **raise awareness**, **highlight risk factors**, and provide **access to support resources**.

* Suicide mortality
* Suicidal thoughts and attempts
* Demographic and socio-economic indicators

All datasets are stored in the `/data` folder.
